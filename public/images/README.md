# 网站Logo图片说明

## 图片位置
所有网站的logo图片应放在 `public/images/` 目录下。

## 图片命名规范
图片文件名应与网站名称对应，使用小写字母和下划线，例如：
- 百度：baidu.png
- 淘宝：taobao.png
- GitHub：github.png

## 图片规格要求
- 格式：PNG、JPG、SVG
- 尺寸：建议 200x200 像素
- 大小：不超过 50KB

## 如何获取Logo图片

1. **官方网站**：访问网站首页，右键保存logo图片
2. **搜索引擎**：搜索"网站名 logo"获取官方logo
3. **设计工具**：使用Canva、Figma等工具制作
4. **图标库**：从iconfinder、flaticon等网站下载

## 临时解决方案

如果暂时没有logo图片，系统会显示网站名称的首字母作为占位符。
